# Consonant Symbols

These are some consonant symbols/modifiers indented for use with the phonograph alphabet.

* glottal stop: `'`
* nasalization: `˜` `˙` `˛`
* labial: `˃`
* coronal: `⁺`
* dorsal: `˂`
* laryngeal: `˝`
* voicing: `ˇ` `亻`
* /h/, aspiration: `˚` `口`
* /j/: `'`
* /l/: `"`
* /r/: `˞` `˹˺˻˼`
* /s/: `ⸯ`
* /w/: `˟`
